About

What is this ?

GWallet is a virtual machine designed to emulate a multi-chain cold wallet. Its main purpose is to integrate browsing and transactions within a self-contained system, effectively mitigating any risks or hacking attempts on managed accounts. The program offers three operational modes. The recommended method is a virtualized machine, which enables users to access their wallets on any operating system supporting VirtualBox technology. The second mode is a real-time option, allowing wallet management directly within a Chrome-family browser as an add-on. This mode is recommended for low-power computers, but it comes with limited usage as it compromises some security features of the virtual console. Lastly, a mobile option is currently under development, which will provide a virtual wallet instance maintaining the same security standards as the PC version, exclusively for Android smartphones.

Why was this created ?

The idea of the wallet manager arose from the limitations we encountered when trying to protect our electronic wallets, and the fact that we couldn't have everything in a single browser while ensuring protection against possible hacks or capital loss. That's why we decided to create this addon and a browser that would unite all the wallets into a single application, while allowing each wallet to be updated without jeopardizing the capital. This way, we achieve a quasi-cold wallet where we can store our capital or simply perform investment actions quickly and securely. And above all, we have everything in one place, being able to access it from our computer regardless of the operating system, or from a TV box or even from a mobile phone.

How to use?

GWallet works simply, by default using the BTC network. Although it is an L2 layer and is in testing, it allows you to test the network and safeguard your funds through a cold storage system. To access your funds, simply click on the wallet located at the bottom left of the application menu. You can create a new wallet or import an existing one. Now, from the crypto menu (accessed by clicking the crypto button), you can install other wallets and import their contents to keep them secure. However, we always recommend creating new wallets that will be used within the manager's interface. Within the application, you will find different areas of applications where you can edit or create documents, accessing Windows and Javascript software without the need for a computer. You will also have quick calculation utilities (often necessary for crypto asset holdings calculations) and even access to a completely free retro gaming section, as well as paid services like Xbox Cloud that have been adapted for keyboard and mouse use. Additionally, you will have access to both paid and free streaming services. The goal is to have everything integrated into a secure system where you can keep your credentials protected.
