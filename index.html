GLASS WALLET


"Unify your cryptocurrencies, simplify your life: The all-in-one cloud-based wallet!"



ABOUT

    The wallet operates as a virtual machine that simulates a multi-chain cold wallet, designed to seamlessly integrate navigation and transactions within an autonomous system, thereby mitigating any risk or hacking attempt on managed accounts. By using a virtual machine compatible with VirtualBox technology, users can carry their wallets across different operating systems that support VirtualBox.

    The concept of Glass Wallet arose from the limitations encountered by the team in protecting electronic wallets. The inability to centralize all wallets within a single browser while ensuring security drove the development of a unified add-on and browser solution. This unified application allows each project to be updated independently without compromising the integrity of individual wallets.

    Enhance your Digital Experience with GWallet:

    • Portfolio Management: Experience effortless access to all wallets with our dedicated Chrome adaptation. GWallet isn't just a wallet; it's a comprehensive work and development companion. Manage your investments effectively with built-in office tools, calculator, text editors, and spreadsheet templates.
    • Gaming: Immerse yourself in over 300 free games while seamlessly managing your wallet networks.
    • Exchange and Liquidity: Swiftly swap tokens, create liquidity for new tokens, or build your own pool without hassle.
    • Network Access: Gain access to various networks including Aptos, Cosmos, Ethereum, SEI, Starknet, SUI, and Venom.
    • Versatile Usage: Carry your wallet securely across different operating systems by installing it on a USB drive. Whether offline or online, access all your assets securely with GWallet.

INSTALLATION

    VirtualBox Installation and OVA Import Manual

  • Installing VirtualBox:

    1. Downloading VirtualBox:
      • Go to the official VirtualBox website at https://www.virtualbox.org/.
      • Click on "Downloads" in the navigation menu.
      • Download the appropriate version of VirtualBox for your operating system (Windows, macOS, Linux, etc.).
    2. Installation:
      • Run the downloaded installation file.
      • Follow the installation wizard instructions.
      • Once the installation is complete, VirtualBox will be ready to use.

  • Importing the GWallet system's OVA file into VirtualBox:

    1. Downloading the GWallet OVA file:
      • Make sure you have the OVA file of the GWallet system that you want to import. It can be a file downloaded from the internet or provided by a developer.
    2. Opening VirtualBox:
      • Open VirtualBox from the start menu or application launcher, depending on your operating system.
    3. Importing the OVA file:
      • In the VirtualBox main window, click on "File" in the top-left corner and select "Import Appliance...".
      • The import wizard will open. Click on "Choose a file" and navigate to the GWallet system's OVA file you want to import.
      • Once the file is selected, click "Next" and then "Import".
      • VirtualBox will start importing the OVA file. This process may take a few minutes, depending on the file size and your system's speed.
    4. Completion:
      • Once the import is complete, you will see the virtual appliance in the list of virtual machines in VirtualBox.
      • Select the newly imported virtual machine and click "Start" to begin using it.

    • That's it! You have successfully installed VirtualBox and imported the GWallet system's OVA file to use it in a virtual machine. Now you can start working with your virtual environment as needed.
